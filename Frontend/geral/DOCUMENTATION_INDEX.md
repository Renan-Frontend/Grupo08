## 🚀 Início Rápido (Copiar & Colar)

### Opção 1: Uso Básico

```jsx
import EditablePipeline from './Components/Opportunities/EditablePipeline';

<EditablePipeline
  initialStages={[
    { id: 1, label: 'Stage 1', done: false },
    { id: 2, label: 'Stage 2', done: false },
  ]}
/>;
```

### Opção 2: Com Salvamento

```jsx
<EditablePipeline
  initialStages={stages}
  onSave={(updated) => console.log('Saved:', updated)}
/>
```

### Opção 3: Completo

```jsx
<EditablePipeline
  initialStages={stages}
  onSave={savePipelineToDatabase}
  title="Meu Processo"
  subtitle="Descrição aqui"
/>
```
